OpenClaw 飞书插件

从社区版升级到官方版 · 完整指南

社区版 → 官方版 流式输出 权限升级

为什么必须升级到官方插件？

对比维度 社区版 官方版
维护主体 社区/个人开发者 飞书开放平台官方
权限能力 仅基础IM消息 OAuth完整授权 · 读写文档/表格/日历
流式输出 ✗ 无/卡顿 ✓ 原生支持 · 边生成边展示
稳定性 易崩溃 官方维护 · 快速迭代

核心价值：流式输出让AI回复边生成边展示，大幅提升交互体验

1

卸载社区版插件

① 备份配置

cp ~/.openclaw/openclaw.json ~/.openclaw/openclaw.json.mybak

② 停止网关服务

openclaw gateway stop

必须先停止网关服务，否则无法卸载插件

1

彻底清理残留配置

③ 卸载插件主体

openclaw plugins disable feishu rm -rf ~/.openclaw/extensions/feishu rm -rf ~/.openclaw/feishu

④ 删除 openclaw.json 中的 feishu 配置

这是卸载中最重要的一步！必须手动删除所有 feishu 相关配置

vim ~/.openclaw/openclaw.json
2

安装飞书官方插件

一行命令完成安装

npx -y @larksuite/openclaw-lark install
  • 扫码创建机器人，权限自动配置
  • 支持关联已有机器人
  • 安装完成后自动重启网关
3

配置流式输出

依次执行以下命令

openclaw config set channels.feishu.streaming false openclaw config set channels.feishu.footer.elapsed true // 开启耗时 openclaw config set channels.feishu.footer.status true // 开启状态展示

最后一步：重启网关

openclaw gateway restart

升级后可以做什么？

📝 AI 会议纪要

飞书会议结束后，AI自动提取要点，生成结构化文档并同步到飞书文档

📊 多维表格自动化

通过AI批量处理多维表格数据，自动填充、筛选、统计

✅ 任务自动管理

发送指令让AI创建飞书任务，分配负责人、设置截止时间

开始升级吧！

告别社区版痛点，享受官方级稳定性和功能体验
流式输出让交互体验直接拉满
