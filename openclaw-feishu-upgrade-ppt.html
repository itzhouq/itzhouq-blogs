⚠️ 重点提醒
必须先彻底卸载社区版，再安装官方版，否则会出现插件冲突、授权失败
第一步

彻底卸载社区版插件

1
备份配置文件
cp ~/.openclaw/openclaw.json ~/.openclaw/openclaw.json.mybak
2
停止网关服务
openclaw gateway stop
3
卸载插件主体
openclaw plugins disable feishu
4
清理残留文件
rm -rf ~/.openclaw/extensions/feishu ~/.openclaw/feishu
5
删除 feishu 配置
手动删除 openclaw.json 中所有 feishu 相关配置
第二步

安装飞书官方插件

npx -y @larksuite/openclaw-lark install
扫码后自动配置权限，重启网关生效

官方命令，安全可靠
自动创建机器人，权限自动配置

第三步

配置流式输出

官方插件默认未开启，手动配置仅需 3 条命令

openclaw config set channels.feishu.streaming false
openclaw config set channels.feishu.footer.elapsed true // 开启耗时
openclaw config set channels.feishu.footer.status true // 开启状态展示

重启网关：openclaw gateway restart
