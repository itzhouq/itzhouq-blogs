快速开始

三步启动使用

  • Step 1 · 安装
    git clone && pip install -r requirements.txt
  • Step 2 · 配置
    hermes init && hermes config set ANTHROPIC_API_KEY xxx
  • Step 3 · 运行
    hermes run · 指定模型：hermes model --provider openrouter --model claude-3.5-sonnet
